# Let's get started

### 1 - Depend on it

##### Add it to your package's pubspec.yaml file

```yml
dependencies:
  fl_chart: ^0.11.0
```


### 2 - Install it

##### Install packages from the command line
```sh
flutter packages get
```

### 3 - Learn it like a charm
